## Item Overview

Custom items in Heirloom use **player head textures** to display unique 3D appearances without requiring a resource pack!

{% hint style="info" %}
**How it works:** Items appear as player heads with custom skin textures, giving them a unique appearance that works in vanilla Minecraft.

## Getting Items (Admin)

Server operators can give custom items using the `/hl give` command:

```
/hl give <item_id> [amount]
```

### Examples

```bash
# Give yourself a pizza
/hl give PIZZA

# Give 64 flour bags
/hl give BAG_OF_FLOUR 64

# Give cheese
/hl give CHEESE 16
```

### Listing All Items

```
/hl list items
```

Shows all registered custom item IDs with their edibility status.

{% hint style="warning" %}
**Permission Required:** The `/hl give` command requires the `heirloom.give` permission (default: OP only).
{% endhint %}

## Recipe Quality System

Many recipes have a **Quality** rating that affects their food value and saturation.

{% hint style="info" %}
**Quality is cumulative!** Adding better ingredients or using special preparation methods increases the final quality rating.
{% endhint %}

### How Quality Works

| Quality | Stars | Bonus |
|---------|-------|-------|

## Special Recipe Mechanics

### Ingredient Properties

Some recipes check for **properties carried over from ingredients**:

{% hint style="info" %}
**Example:** Pizza quality increases if the dough contained yeast (brown mushroom), even though yeast isn't a direct pizza ingredient!
{% endhint %}

### Return Items

Certain ingredients return their container after cooking:

| Ingredient | Returns |
|------------|---------|
| Milk Bucket | Bucket |
| Water Bucket | Bucket |
| Glass Bottle (in recipes) | Glass Bottle |
| Honey Bottle (in recipes) | Glass Bottle |

### Consume Returns

Some foods return items when eaten:

| Food | Returns When Eaten |
|------|-------------------|
| Jam | Glass Bottle |
| Salad | Bowl |
| Tomato Soup | Bowl |
